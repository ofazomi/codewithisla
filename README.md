* by mary rose cook

An interpreter for Isla in JavaScript.

## Run the tests

Install Node.js and npm.

Install the node dependencies

    $ cd path/to/islajs
    $ npm install

Run the tests

    $ cd path/to/islajs
    $ node_modules/jasmine-node/bin/jasmine-node spec/
